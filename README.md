# Responsive Blog Card Layout

A responsive blog-style card layout built using **HTML5** and **CSS3**, showcasing articles in a clean grid-based design with smooth hover effects and consistent alignment across devices.

## Features
- Responsive layout using CSS Grid  
- Card-based UI with hover interactions  
- Semantic HTML and clean CSS structure  
- Mobile, tablet, and desktop friendly  

## Tech Stack
HTML5, CSS3
